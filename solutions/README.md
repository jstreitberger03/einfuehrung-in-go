# Lösungen: Einführung in Go

Dieser Ordner enthält Musterlösungen zu den Übungen am Ende jedes Kapitels.

## Struktur

```
solutions/
├── 00_intro/          # Lösungen zu Kapitel 0
├── 01_basics/         # Lösungen zu Kapitel 1
├── 02_control_flow/   # Lösungen zu Kapitel 2
...
```

> **Für Lernende:** Versuche die Übungen zuerst selbst! Schau erst in die Lösungen, nachdem du es wirklich versucht hast.

> **Für Beitragende:** Lösungen sind als Pull Requests willkommen. Verwende das gleiche Format wie die Kurs-Kapitel.
